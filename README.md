# Tensorfold.ai — Time Tracker

## 🚀 Deploy to Vercel

### Option A — Vercel CLI
```bash
npm install
npx vercel
```

### Option B — GitHub
1. Push to GitHub
2. Go to vercel.com/new → Import repo → Deploy ✅

## 💻 Run Locally
```bash
npm install
npm run dev
# Open http://localhost:5173
```
